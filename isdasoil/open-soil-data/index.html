Open access soil data for Africa

About our dedication to open access soil data.

As part of our commitment to open access data, we are maintaining a soil analysis dataset for Africa. Our intention is to bring together as much soil data as possible in an analysis-ready format. We use this dataset to train the models that create the iSDAsoil maps. We plan to regularly update this resource with additional soil data.

 

Contributing data

 

We are maintaining a GitHub repository at https://github.com/iSDA-Africa/open-soil-data. We welcome any pull requests contributing additional data. Alternatively you can get in touch via email.

 

How can I access the data?

 

In addition to hosting on Github, csv files containing the data and metadata can be downloaded by following the respective links. New versions containing additional data may be released in future.

 

Citing this dataset

 

This dataset is available under a CC-BY-4.0 licence. Please cite using the following DOI: https://doi.org/10.17605/OSF.IO/A69R5
